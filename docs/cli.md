# CLI Usage

```console
$ hitfactorpy-graphql-server --help
$ python -m hitfactorpy-graphql-server --help
```

## `--version`

Prints the package version and exits.

```console
$ hitfactorpy-graphql-server --version
0.0.1
```

## `run`

Shortcut for `uvicorn.run`.

```console
$ hitfactorpy-graphql-server run
```
